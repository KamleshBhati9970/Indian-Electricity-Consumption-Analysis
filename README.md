# Indian Electricity Consumption Analysis: Quantifying the Lockdown Effect (2019–2020)

## Objective

This project provides a **data-driven deep dive into India’s electricity consumption patterns from January 2019 to December 2020**. By analyzing **16,500+ daily records**, the system aims to:

- **Quantify the Lockdown Impact:** Measure the exact energy dip during the peak lockdown period (**March–June 2020**).
- **Identify Regional Trends:** Map how different states responded to the crisis based on their **industrial or agricultural dependencies**.
- **Support Future Planning:** Build a foundation for understanding **regional energy resilience** and **national power stability**.

## Key Insights Delivered

### The Lockdown Shock
Visualizing the **March 2020 electricity demand collapse** compared to the **stable 2019 baseline**.

### Regional Resilience
Identifying which **geographical power regions (North, West, South, etc.)** recovered the fastest.

### Recovery Index
A **custom metric** designed to track the **speed of recovery of industrial electricity demand** after the lockdown.
